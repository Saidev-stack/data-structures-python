# data-structures-python
A Python project implementing linked lists, stacks, queues, trees, binary trees and binary search trees from scratch.

## Implemented Data Structures
- Linear Linked Lists
- Stacks
- Queues
- General Trees
- Binary Trees
- Binary Search Trees

## Features
- Creation of data structures
- Insertion and deletion operations
- Search algorithms
- Tree height calculation
- Node and leaf counting
- Binary Search Tree validation

## How to run
Run the main program:
python main.py

## Technologies
- Python 3
- Standard Python library only
